=================
Opkg 套件庫使用說明
=================

版本: 0.1.8
檔案: opkg_0.1.8_tomato-k26_mips32.ipk
功能: 套件庫管理

軟硬體需求:
	1. 已安裝Tomato K26(核心)等衍生版本韌體(泰迪熊, 土司超人...)的路由器.
	2. 已安裝Optware ddwrt套件庫.

安裝(Telnet/SSH):
	# ipkg-opt install http://twtomato.googlecode.com/files/opkg_0.1.8_tomato-k26_mips32.ipk
	# opkg update
	# opkg list
